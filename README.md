# DigitalAuth

DigitalAuth is a web application built with Next.js, TypeScript, Tailwind CSS, MongoDB, and Editor.js. It allows multiple users to collaborate on shared documents and aims to incorporate digital authentication for added security in the future.

## Technologies Used

- Next.js
- TypeScript
- Tailwind CSS
- MongoDB
- Editor.js

## Features

- **Multi-user Collaboration:** Users can collaborate on shared documents in real-time.
- **MongoDB Integration:** Utilizes MongoDB for data storage.
- **Editor.js Integration:** Employs Editor.js for document editing capabilities.
- **Future Plans:** Implementation of digital authentication for enhanced security.

## Getting Started

To run this program locally, follow these steps:

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/Saurabhn16/DigitalAuth
   ```

2. Navigate into the project directory:

   ```bash
   cd DigitalAuth
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```

5. Access the application at [http://localhost:3000](http://localhost:3000) in your web browser.

Ensure that MongoDB is installed and running on your machine or that you have access to a MongoDB instance where the application can connect for data storage.
